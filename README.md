# Проект автотестов для API Avito

## Структура проекта

- **/tests:** Тестовые сценарии.
- **/test\_data:** Тестовые данные в формате JSON.
- **/utils:** Вспомогательные утилиты для API клиента и провайдера данных.
- `requirements.txt`: Список зависимостей.
- `pytest.ini`: Конфигурация Pytest.
- `README.md`: Данное руководство.

## Инструкция по установке и запуску

### Предусловия

- Установленный Python 3.8+
- Установленный pip

### Установка зависимостей

1.  **Склонируйте репозиторий:**
    ```bash
    git clone https://github.com/Maya010505/qa-avito-internship-autumn2025.git
    cd task2
    ```

2.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

### Запуск тестов

Для запуска всех тестов выполните следующую команду в корневой директории проекта:

```bash pytest 
    python -m pytest
```
